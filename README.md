# Renzie’s Fairy Eggs

Mobile-first local operations app for Renzie’s Fairy Eggs in Cape Girardeau, Missouri.

## What it does

This is a plain HTML/CSS/JavaScript app with no backend and no external dependencies. It stores data in the browser using localStorage.

Core features:
- Dashboard / Today view
- Customers
- Products and inventory
- Egg collection tracking
- Orders with delivery fee logic
- Inventory ledger
- Purchases / expenses
- Reports
- JSON backup export/import
- Printable inventory stock sheets, packing sheets, customer list, and sales report

## How to run locally

Open `index.html` directly in any browser.

## Important backup warning

The app saves data in browser localStorage. Use the JSON export button often to back up business data.
